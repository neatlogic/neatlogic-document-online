# 虚拟化平台采集
虚拟化采集主体包括：数据中心、集群、集群与物理机关系、集群与操作系统关系、存储关系等，同时也是对[操作系统采集](2.操作系统采集.md)的数据进行关系计算和数据补充。

或API方式抓取和采集数据。
