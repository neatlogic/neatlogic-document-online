# 容器平台采集
目前主要是针对k8s容器管理平台的数据采集，主体包括：deployments、ReplicaSets、Pods、nodes等数据采集，同时也是对[操作系统采集](2.操作系统采集.md)的数据进行关系计算和数据补充。
